// turbo-all

8GB RAM 맥 환경의 쾌적함을 유지하기 위한 시스템 자동 세정 워크플로우입니다.

1. **최적화 엔진 실행**
   - `python3 antigravity_auto_clean.py` 명령어를 통해 시스템 전반을 청소합니다.

2. **메모리 상태 확인**
   - 장기 기억(`data/team_memory.json`)에 최적화 시간이 기록되었는지 확인합니다.

3. **추가 권장 작업**
   - 작업이 완료된 후 Antigravity IDE를 한 번 재시작하면 더욱 효과적입니다.
