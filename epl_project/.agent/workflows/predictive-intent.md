# Predictive Intent System

## 목적
사용자의 다음 요청을 AI가 예측하여 선제적으로 준비

## 예측 로직

### 1. 패턴 학습
```python
# 사용자 요청 히스토리 분석
request_history = [
    "EDA 분석",
    "상관관계 분석",
    "머신러닝 모델",
    # → 다음은 80% 확률로 "모델 해석(SHAP)" 요청 예상
]
```

### 2. 선제적 제안 예시
```
사용자: "머신러닝 분석 완료했어요"

Antigravity 응답:
"✅ RandomForest 모델 완료! (정확도 40%)

💡 다음 단계 제안:
1. SHAP으로 모델 해석 (추천 ⭐)
2. Causal AI로 인과관계 분석
3. 하이퍼파라미터 튜닝으로 정확도 향상

1번을 바로 진행할까요?"
```

### 3. 컨텍스트 기반 예측
- **시간대**: 오전 → 데이터 로드/EDA, 오후 → 고급 분석
- **프로젝트 단계**: 초기 → 탐색, 중기 → 모델링, 후기 → 보고서
- **최근 오류**: 패키지 설치 실패 → 다음엔 가상환경 제안

## 자동화
- 모든 분석 완료 후 자동으로 다음 단계 3가지 제안
- 사용자 승인 시 즉시 실행
