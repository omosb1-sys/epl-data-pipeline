# User Guide & Skill Registry

이 파일은 안티그래비티 시스템에서 사용 가능한 **절차적 지식(Skill)**의 목록을 관리하는 레지스트리입니다.
사용자는 필요에 따라 해당 스킬을 호출하여 에이전트의 페르소나를 전환할 수 있습니다.

## 🛠️ Available Skills (설치된 스킬 목록)

| 스킬명 (Skill Name) | 파일 위치 | 설명 (Description) |
| :--- | :--- | :--- |
| **Causal Inference Expert** | `.agent/skills/causal_inference_expert.md` | `DoWhy`와 `CausalImpact`를 활용해 데이터의 진짜 인과관계를 밝혀내는 통계 전문가입니다. 단순 상관관계를 넘어선 '원인 분석' 시 사용하세요. |
| **Polars High-Perf Engineer** | `.agent/skills/polars_performance.md` | 8GB RAM 환경에서 `Polars LazyFrame`과 `Streaming`을 활용해 대용량 데이터를 처리하는 고성능 엔지니어링 가이드입니다. |
| **Dashboard Architect** | `.agent/skills/dashboard_architect.md` | `Streamlit`과 `PyGWalker`를 활용해 정적 분석 결과를 즉시 '인터랙티브 웹 앱'으로 변환하는 시각화 아키텍트입니다. |
| **Penpot Design Expert** | `.agent/skills/penpot_expert.md` | Figma 대안인 오픈소스 디자인 툴 'Penpot'의 활용 가이드입니다. |

## 🚀 How to Use (사용법)

*   **스킬 장착**: 에이전트에게 "인과 추론 스킬 써서 분석해줘" 또는 "Polars 최적화 모드로 짜줘"라고 요청하세요.
*   **스킬 업데이트**: 새로운 기술이 필요하면 `Skill Seeker`에게 "X 기술 스킬 만들어줘"라고 요청하세요.
